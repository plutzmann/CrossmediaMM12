Installation

Before installing theme, you need to setup your Wordpress. If you don't know how to install wordpress Please go to the General Guides section there has First steps video and other tons of helpful tutorials are there.

Install theme via FTP

Step 1 – Log into your hosting space via an FTP software(Example: FileZilla etc)

Step 2 – Unzip the Mixing-version-1.zip file and ONLY use the extracted Mixing theme folder

Step 3 – Upload the extracted Mixing theme folder into wp-content > themes folder

Step 4 – Activate the newly installed theme. Go to Appearance > Themes and activate it.

Step 6 – Then you will get a notification to activate the required plugins: Layer Slider and CF7. Follow the steps to install and activate.

Common Errors Uploading theme

If you get the “Are You Sure You Want To Do This” message when installing Mixing.zip file via WordPress, it means you have an upload file size limit. Install the theme via FTP if this happens, or call your hosting company and ask them to increase the limit.

Updating Theme

Our theme is keep improving for fixing bugs and other related things. And you need to keep update your theme to new versions.

Update via FTP

Step 1 – Go to “wp-content > themes” location and backup your “Mixing” theme folder by saving it to your computer, or you can choose to simply delete it. Your content will not be lost.

Step 2 – Retrieve the “Mixing-version-1.zip” file from your new Themeforest download and extract the file to get the “Mixing” theme folder.

Step 3 – Then simply drag and drop the new “Mixing” theme folder into “wp-content > themes” location. Choose to “Replace” the current one if you did not delete it.

Update via Wordpress Dashboard

Step 1 – Go to your WP dashboard area and Please make sure you are backup your current theme if you edited theme files, then after you make sure delete the current theme.

Step 2 – Retrieve the “Mixing-version-1.zip” file from your new Themeforest download and upload it on Appearance-> Themes section.

Step 3 – Then activate the theme again.

Header Types

There are 5 different types are there and you can change it very easy.

  • Title and Subtitle - Displaying title and subtitle and breadcrumps
  • Slider - Displaying Slider on any page. Mostly used on Home Page etc
  • Featured Image - Displaying your set Featured Image
  • Google Map - Displaying Google Map on any page. Mostly used on Contact Page

Each sections option will be displayed which Header types you are chosen.
